# Lista de Tarefas

![Print da tela](./src/assets/images/to-do%20list.jpg)

## Modo de usar

### Git Bash (SSH)

     $ git clone git@github.com:JoaoPauloPereirax/to-do-list-react-app.git

### Terminal (Dentro da pasta)

     $ npm install

### Rodando em um servidor local (http://localhost:3000)

     $ npm start
